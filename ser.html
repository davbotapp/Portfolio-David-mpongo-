Les Balises HTML essentielles

  • <!DOCTYPE html> : Définit le type de document HTML5.
  • <html> : Racine de la page.
  • <head> : Métadonnées (titre, charset, CSS).
  • <title> : Titre de la page.
  • <meta charset="UTF-8"> : Encodage des caractères.
  • <link> : Lien vers CSS externe.
  • <style> : CSS interne.
  • <body> : Contenu visible de la page.
  • <h1> à <h6> : Titres (importance décroissante).
  • <p> : Paragraphe.
  • <a> : Lien hypertexte.
  • <img> : Image.
  • <ul> : Liste non ordonnée.
  • <ol> : Liste ordonnée.
  • <li> : Élément de liste.
  • <div> : Conteneur bloc.
  • <span> : Conteneur inline.
  • <form> : Formulaire.
  • <input> : Champ de formulaire.
  • <button> : Bouton.
  • <textarea> : Zone texte.
  • <label> : Étiquette pour input.
  • <table> : Tableau.
  • <tr> : Ligne tableau.
  • <td> : Cellule tableau.
  • <th> : En-tête tableau.
  • <header> : En-tête page/section.
  • <footer> : Pied de page.
  • <nav> : Navigation.
  • <section> : Section.
  • <article> : Article.
  • <aside> : Contenu secondaire.
  • <strong> : Texte important (gras).
  • <em> : Texte emphatique (italique).
  • <br> : Saut de ligne.
  • <hr> : Ligne horizontale.
  • <iframe> : Cadre embarqué.
  • <audio> : Audio.
  • <video> : Vidéo.
  • <source> : Source audio/vidéo.
  • <canvas> : Dessin 2D/3D.
  • <script> : JavaScript.
  • <main> : Contenu principal.
  • <picture> : Image responsive.
  • <details> : Contenu déroulant.
  • <summary> : Titre déroulant.
  • <mark> : Texte surligné.
  • <progress> : Barre de progression.
  • <time> : Date/heure.
  • <meta name="viewport" content="width=device-width, initial-scale=1"> : Responsive.
  • <style> : CSS interne.
  • <details> : Détails déroulants.
  • <abbr> : Abréviation.
  • <code> : Bloc code.
  • <pre> : Texte préformaté.
  • <blockquote> : Citation longue.
  • <cite> : Citation source.
  • <small> : Texte secondaire.
  • <sub> : Indice.
  • <sup> : Exposant.
  • <del> : Texte supprimé.
  • <ins> : Texte inséré.

Exercice pratique

Écris un petit code HTML utilisant au moins 3 balises vues ci-dessus (ex: <h1>, <p>, <a>). Clique sur "Valider" pour voir le rendu.
